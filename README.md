# LEE DO SEON Portfolio Website

이 프로젝트는 LEE DO SEON의 포트폴리오 웹사이트입니다.

## 기술 스택

- HTML5
- CSS3
- JavaScript
- Node.js
- Express

## 설치 및 실행

1. 프로젝트 클론

```bash
git clone [repository-url]
cd portfolio
```

2. 의존성 설치

```bash
npm install
```

3. 개발 서버 실행

```bash
npm run dev
```

4. 프로덕션 서버 실행

```bash
npm start
```
